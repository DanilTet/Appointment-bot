# Appointment-bot

Telegram-бот для автоматической и ручной записи пациентов на медицинские приемы (эндоскопия, консультации, УЗИ) к врачу-эндоскописту Олегу Тетернику, с интеграцией с Google Таблицами и базой данных Supabase.

## Основной функционал

*   **Интерактивная запись для пациентов**: Быстрый выбор услуг, дат и свободного времени.
*   **Синхронизация с Google Sheets**: Автоматический импорт ручных записей (включая записи других врачей, таких как Данило и Калашников) и экспорт подтвержденных заявок обратно в таблицу.
*   **Визуальное расписание**: Генерация красивых PNG-карточек расписания на завтра/сегодня для врача прямо в чате вместо длинных текстовых сообщений.
*   **Мониторинг приемов в реальном времени**: Отслеживание этапов приема («Запланировано» -> «У анестезиолога» -> «Выполняется» -> «Выполнено») с уведомлением администраторов.
*   **Напоминания**: Каскадные автоматические напоминания пациентам за 24 часа, утром в день приема (06:00) и за 1 час с возможностью подтвердить или отменить визит.
*   **Система отзывов**: Модерируемые отзывы с оценкой от 1 до 5 звезд и удобной пагинацией прямо в Telegram.
*   **Интеграция с веб-сайтом**: Отслеживание лидов с внешнего сайта через таблицу `site_leads` в Supabase и мгновенная отправка уведомлений администраторам.

## Архитектура проекта

```
Appointment-bot/
├── config.py             # Настройки бота, константы и маппинги услуг
├── main.py               # Точка входа для запуска Telegram-бота
├── requirements.txt      # Зависимости Python проекта
├── credentials.json      # Ключи доступа Google API (Service Account)
├── services/
│   ├── db.py             # Работа с базой данных Supabase
│   ├── sheets.py         # Интеграция с Google Sheets API
│   └── renderer.py       # Генерация красивых PNG расписаний на Pillow
├── tg_bot/
│   ├── states.py         # FSM-состояния пользователей (Запись, Отзывы)
│   ├── keyboards.py      # Генерация клавиатур для админов и клиентов
│   └── handlers/
│       ├── patient.py    # Логика клиента (Запись, Личный кабинет, Отзывы)
│       └── admin.py      # Панель администратора (Подтверждение приемов, статистика)
└── workers/
    └── tasks.py          # Фоновые процессы (напоминания, синхронизация с Sheets, утренние отчеты)
```

## Установка и запуск

1. **Клонируйте репозиторий и настройте виртуальное окружение**:
   ```bash
   git clone <URL_репозитория>
   cd Appointment-bot
   python -m venv .venv
   source .venv/bin/activate  # На Windows: .venv\Scripts\activate
   pip install -r requirements.txt
   ```

2. **Заполните файл конфигурации `.env`**:
   Создайте файл `.env` в корневой папке и укажите необходимые переменные:
   ```env
   TELEGRAM_BOT_TOKEN=your_token
   SUPABASE_URL=your_supabase_url
   SUPABASE_KEY=your_supabase_anon_or_service_role_key
   SCHEDULE_SHEET_ID=your_google_sheet_id
   ```

3. **Запуск проекта**:
   *   Запуск бота:
       ```bash
       python main.py
       ```

## Визуальное расписание (Pillow)
В проекте используется модуль `services/renderer.py`, который динамически собирает изображение расписания:
*   Карточки процедур используют цветовую индикацию.
*   Специальные исследования с наркозом подсвечиваются ярким красным цветом (`💉 Наркоз`) для повышенного внимания врача.
*   Если генерация картинки дает сбой, бот автоматически переключается на стандартный текстовый отчет.
